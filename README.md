---

## Prompt 2

"Quais são as melhores práticas para escrever prompts?"

Resultado:

Foi explicado uma série de técnicas e métodos para diminuir as chances de alucinações:

- permissão para admitir ignorância
- Ajuste de temperatura do modelo
- inclusão de documentos e dados
- solicitar verificação de dados
- Utilizar formatos estruturados
- Aplicação de cadeia de pensamento
- técnica de consistência própria

# 📖 Miniguia de Estudos

## O que é Engenharia de Prompts?

É a prática de elaborar instruções claras para obter respostas melhores de modelos de IA.

---

## Boas práticas

- Seja específico.
- Informe o contexto.
- Defina o formato da resposta.
- Utilize exemplos.
- Faça refinamentos.

---

# 📚 Glossário

**LLM**
Modelo de Linguagem de Grande Escala.

**Prompt**
Texto enviado para a IA.

**Contexto**
Informações adicionais fornecidas para melhorar a resposta.

**Temperatura**
Parâmetro que controla a criatividade das respostas.
